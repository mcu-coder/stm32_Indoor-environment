
STM32室内环境监测系统功能如下：
1.	使用STM32F103C8T6单片机作为主控单元
2.	光敏电阻采集环境光强度，温度传感器采集温度，光电红外传感器采集人体有无
3.	OLED显示屏实时显示当前环境信息
4.	光电红外检测到有人，可根据环境光照强度控制照明灯开关，根据环境温度情况控制风扇风速（低/高风速）和加热/制冷的操作
5.	按键可设置阈值
6.	可以通过蓝牙接入APP，APP实现数据上传与控制下发的功能


2.材料清单

•	STM32F103C8T6单片机

•	OLED 屏幕

•	DS18B20温度传感器

•	光电红外传感器

•	光敏传感器

•	BT04A蓝牙模块

•	TB6612电机调速模块

•	继电器

•	风扇模块

•	PTC加热片
